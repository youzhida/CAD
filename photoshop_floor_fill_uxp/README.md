# Floor Fill Auto

这是给当前彩平 PSD 模板做的第一版 Photoshop UXP 插件。

## 这版只做什么

- 自动分析 PDF 智能对象线稿
- 自动更新 `图层 2`
  - 外圈背景灰
  - 房间浅灰
- 自动更新 `图层 7`
  - 只同步房间区域像素
  - 保留你现成模板里的 `Inner Shadow`

## 这版明确不做什么

- 不做更深的装饰灰块
- 不做蓝色水池 / 浴缸
- 不做任意风格图纸的通用识别

## 当前文档约定

- 文档里已经有一层 PDF 智能对象线稿
- 文档里优先已经有 `图层 2` 和 `图层 7`
- 如果没有 `图层 7`，插件会新建，但新建层不会自动带你的现成阴影样式

## 加载方式

1. 打开 Adobe UXP Developer Tool
2. 选择 `Add Plugin`
3. 指向当前目录 `F:\ys\photoshop_floor_fill_uxp`
4. 在 Photoshop 里打开面板 `Floor Fill Auto`
5. 点一次 `生成图层2 + 图层7`

## 已知限制

- 第一版是按你现在这套模板假设写的
- 如果 PDF 线稿层名字变化很大，或者模板层名不是 `图层 2` / `图层 7`，需要再调一下
- 现在的房间识别是“封闭白区 + 面积过滤 + 小洞补平”，所以非常规户型还需要再拿样本继续修
