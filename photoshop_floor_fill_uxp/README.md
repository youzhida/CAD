# Floor Fill Auto

这是给当前彩平 PSD 模板做的第一版 Photoshop UXP 插件。

## 这版只做什么

- 自动分析 PDF 智能对象线稿
- 自动在 PDF 线稿下方补一层白底 `PDF白底`
- 自动更新 `图层 2`
  - 整页背景灰
  - 只保留主户型白底
  - 主户型内默认都按灰地面处理
  - 文字 / 门扇 / 门洞弧线默认跟着灰地面走
  - 床 / 家具 / 沙发 / 桌子这类内部闭合大白块会在原始线稿分辨率上单独扣回白
  - 自动把图框 / 尺寸 / 标题栏吃进背景灰
- 自动更新 `图层 7`
  - 只同步浅灰地面区像素
  - 每次运行都会强制同步默认 `Inner Shadow`

## 这版明确不做什么

- 不做更深的装饰灰块
- 不做蓝色水池 / 浴缸
- 不做任意风格图纸的通用识别

## 当前文档约定

- 文档里已经有一层 PDF 智能对象线稿
- 文档里优先已经有 `图层 2` 和 `图层 7`
- 如果没有 `PDF白底`，插件会自动补到 PDF 线稿下面
- 如果没有 `图层 7`，插件会新建；无论新旧，都会强制同步默认内阴影
- 图层顺序固定为：PDF 线稿最上，其次 `图层 7`，再是 `图层 2`，最下面是 `PDF白底`

## 加载方式

1. 打开 Adobe UXP Developer Tool
2. 选择 `Add Plugin`
3. 指向当前目录 `D:\cc\photoshop_floor_fill_uxp`
4. 在 Photoshop 里打开面板 `Floor Fill Auto`
5. 点一次 `生成图层2 + 图层7`

## 已知限制

- 第一版是按你现在这套模板假设写的
- 如果 PDF 线稿层名字变化很大，或者模板层名不是 `图层 2` / `图层 7`，需要再调一下
- 现在的房间识别是“主户型范围 + 封闭白区 + 面积过滤 + 家具内嵌排除”，所以非常规户型还需要再拿样本继续修
